## ☑️ 실행 방법

- **Clone Project**
  ```bash
  git clone https://github.com/yunojang/dabeeo_test.git
  ```
- **Package install (택 1)**
  ```bash
  yarn
  npm install
  ```
- **Run server (택 1)**
  ```bash
  yarn start
  npm start
  ```

## ✅ 구현 사항(요구 사항)

- map 이미지를 사이즈 조절 없이 1024 X 728 div에 배치한다.
- 드래그를 통해 움직이고, 이미지 끝으로 가면 더이상 드래그 할 수 없다.
- 우클릭으로 마커를 생성한다.
- 우측 상단 리셋 버튼을 누르면 마커가 모두 사라진다.
